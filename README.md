# 🎮 Hangman Game (Python)

A simple and interactive **terminal-based Hangman game** built using Python.
Guess the word letter by letter before you run out of lives!

---

## 🚀 Features

- 🎯 Random word selection
- 💀 ASCII Hangman visuals
- 🔤 Input validation
- ❤️ Lives system
- 📜 Tracks used letters

---

## ▶️ How to Run

1. Clone the repository:

```
git clone https://github.com/debmallardasgupta/FIRST-REPO.git
cd FIRST-REPO
```

2. Run the game:

```
python main.py
```

---

## 🧠 How It Works

- A random word is selected from the word bank
- You guess one letter at a time
- Each wrong guess reduces your lives
- Game ends when:
  - You guess the word ✅
  - You run out of lives ❌
